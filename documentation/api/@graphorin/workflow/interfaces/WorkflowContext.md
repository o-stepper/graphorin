# Interface: WorkflowContext\&lt;TState\&gt;

Defined in: packages/workflow/src/types.ts:83

**`Stable`**

Argument shape passed to a node's `run(...)` callback. The runtime
supplies a fresh context per task with the writer-bound `emit`
helper plus per-task identifiers.
