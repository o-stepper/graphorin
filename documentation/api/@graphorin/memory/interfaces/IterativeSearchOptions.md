# Interface: IterativeSearchOptions

Defined in: [packages/memory/src/tiers/semantic-memory.ts:330](https://github.com/o-stepper/graphorin/blob/main/packages/memory/src/tiers/semantic-memory.ts#L330)

Per-call options for [SemanticMemory.searchIterative](/api/@graphorin/memory/classes/SemanticMemory.md#searchiterative) (P2-4) -
the gated grade-then-reformulate loop. Extends [FactSearchOptions](/api/@graphorin/memory/interfaces/FactSearchOptions.md)
(every base option applies to each retrieval pass); `topK` doubles as
the cap on the accumulated result count.
